## Instalation

It has been tested with Python 3.6.8.

For instalation, you will install the requirements **in path of friday**.

    pip install -r requirements.txt

To run service use:

    python friday.py

Enter in browser address. Like **ip:port\bot**
  
Have fun!!! =)
